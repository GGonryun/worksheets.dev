# Code Sandbox

This is a playground for the `@worksheets/sdk` package. It is a Code Sandbox
that you can use to try out the SDK and see how it works.

## Requirements:

- This sandbox requires a [Worksheets](https://worksheets.dev) account and API Key.
  - [Learn more about how to get an API key](/getting-started).
  - No credit card required.
  - 100 free API calls per month.

## Playground

For more information on the SDK, see the [README](https://www.npmjs.com/package/@worksheets/sdk).
